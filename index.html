➕ Adicionar Novo Colaborador
Para adicionar um novo colaborador, acesse o painel do Supabase:

1. Acesse supabase.com/dashboard → projeto tradutor-sptraducoes
2. Menu → Authentication → Users → Add user → Create new user
3. Preencha e-mail e senha e marque "Auto Confirm User"
4. No SQL Editor rode:

INSERT INTO users (id, name, email, role, active)
SELECT id, 'Nome Completo', '[email protected]', 'operator', true
FROM auth.users WHERE email = '[email protected]';
